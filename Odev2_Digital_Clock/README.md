# Ödev 2 – LCD Saat Uygulaması
Bu proje, Tiva C serisi mikrodenetleyici kullanılarak 16x2 LCD ekran üzerinde anlık saatin (SS:DD:SN) formatında gösterilmesini amaçlamaktadır. Zaman bilgisi, mikrodenetleyici zamanlayıcıları (timer) kullanılarak yazılım tabanlı olarak oluşturulmuştur.
Uygulama kapsamında LCD sürme işlemleri ve zaman hesaplamaları bir arada kullanılarak gerçek zamanlı saat mantığı temel seviyede uygulanmıştır.
