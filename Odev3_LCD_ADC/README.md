# Ödev 3 – LCD Üzerinde ADC Değer Okuma
Bu proje, Tiva C serisi mikrodenetleyici kullanılarak potansiyometre üzerinden alınan analog verinin ADC birimi ile dijital değere dönüştürülmesini ve 16x2 LCD ekran üzerinde görüntülenmesini amaçlamaktadır. 
Uygulama kapsamında ADC yapılandırma ayarları yapılmış, elde edilen sayısal veri string formatına dönüştürülerek LCD üzerinde okunabilir şekilde gösterilmiştir.
